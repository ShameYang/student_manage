## 项目简介

- 基于 java + mysql + servlet + jsp 的学生管理系统
  - 该项目是我个人学习 JavaWeb 的练手小项目
  - 目前处于初学阶段，学完每个知识点后都会进行改进
  - 代码会有冗余等问题，之后也会改进
  - 注意：前端页面能看就行，没有进行美化...
  
- 实现功能
  - 详情
    - 查询某个同学的详细信息
  - 修改
    - 修改学生信息
  - 删除
    - 删除某个同学的信息
  - 新增学生
    - 添加学生信息
  - 登录
    - 登录成功，跳转到学生基本信息页面
    - 登录失败，跳转到错误页面，提示重新登录
  - 退出系统
    - 跳转到登录页面
